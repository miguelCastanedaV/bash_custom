# Utilidades para consola linux

Para instalar directamente desde GitHub:
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/miguelcastanedaV/bash_custom/main/install.sh)
```

O si prefieres descargar primero y luego ejecutar:
```bash
curl -fsSL https://raw.githubusercontent.com/miguelcastanedaV/bash_custom/main/install.sh -o install.sh && chmod +x install.sh && ./install.sh
```
